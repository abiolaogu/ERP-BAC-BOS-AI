## Documentation Standards

All documentation in this suite follows these standards:

- **Markdown Format**: All documents in Markdown (.md)
- **Table of Contents**: Each guide includes a TOC
- **Code Examples**: Syntax-highlighted code blocks
- **Cross-References**: Links between related documents
- **Version Info**: Version and last updated date
- **Target Audience**: Clearly specified for each guide
- **Screenshots**: Placeholders for visual aids (marked with `[SCREENSHOT: ...]`)
