- üîí **Secure**: End-to-end encryption
- üì± **Cross-Platform**: Web, mobile, desktop

**Minimum**
- Browser: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+

## Starting a Meeting

### Instant Meeting

#### Quick Start
1. Open NEXUS Hub
2. Click **"Start Meeting"** button
3. Or go to `nexus.yourcompany.com/meet`
4. Click **"New Meeting"**
5. Meeting starts immediately

#### From Calendar
1. Open NEXUS Calendar
2. Create event
3. Click **"Add video conferencing"**
4. Select **"NEXUS Meet"**
5. Meeting link added automatically
6. Attendees can join from calendar invite

### Schedule a Meeting

#### From NEXUS Meet
1. Click **"New Meeting"** ‚Üí **"Schedule in Calendar"**
2. Set meeting details:
   - **Title**: Meeting name
   - **Date**: Meeting date
   - **Time**: Start and end time
   - **Guests**: Add attendees
   - **Description**: Agenda and notes
3. Click **"Save"**
4. Invites sent to attendees
5. Meeting link in calendar event

#### Recurring Meetings
1. When scheduling, click **"Does not repeat"**
2. Choose recurrence:
   - Daily
   - Weekly (select days)
   - Monthly
   - Custom
3. Set end date or number of occurrences
4. Same meeting link for all instances

### Meeting Settings

Before starting, configure:

**Video**
- ‚úÖ On by default
- ‚ùå Off by default (voice only)

**Audio**
- ‚úÖ Microphone on
- ‚ùå Microphone muted

**Entry**
- ‚úÖ Auto-admit all
- ‚ùå Waiting room (host admits)
- ‚ùå Knock to enter

**Recording**
- ‚úÖ Allow recording
- ‚ùå Prevent recording

## Joining Meetings

### Join via Link

1. **Click meeting link** from:
   - Email invitation
   - Calendar event
   - Chat message
   - Shared link
2. **Choose how to join**:
   - NEXUS Meet web app (no download)
   - Desktop app (if installed)
   - Mobile app (iOS/Android)
3. **Allow permissions**:
   - Camera access
   - Microphone access
4. **Preview** your video/audio
5. Click **"Join now"**

### Join via Meeting Code

1. Open NEXUS Meet
2. Click **"Join with code"**
3. Enter meeting code (e.g., abc-defg-hij)
4. Click **"Join"**

### Guest Access (No Account Required)

Guests can join without NEXUS account:
1. Click meeting link
2. Enter name
3. Click **"Ask to join"**
4. Wait for host to admit
5. Join meeting

### Join from Phone

**Dial-in option**:
1. See dial-in number in invite
2. Call the number
3. Enter meeting ID
4. Press # to join
5. Audio-only participation

## Screen Sharing

### Share Your Screen

1. Click **"Share screen"** button (üñ•Ô∏è)
2. Choose what to share:
   - **Entire screen**: Everything on your screen
   - **Application window**: Specific app only
   - **Chrome tab**: Specific browser tab
3. Click **"Share"**
4. Screen sharing starts

### Sharing Options

**Entire Screen**
- Shows everything: all windows, taskbar, notifications
- ‚ö†Ô∏è Close sensitive apps first
- Good for: Demos, tutorials

**Application Window**
- Shows only selected application
- Safer: notifications hidden
- Good for: Specific app demos

**Chrome Tab**
- Shows only one browser tab
- Can include audio
- Good for: Presentations, videos

**Advanced Features**
- **Share audio**: Include computer sound
- **Optimize for video**: Better for video playback
- **Optimize for text**: Better for documents/code

### Control Shared Screen

While sharing:
- **Pause**: Temporarily pause sharing
- **Resume**: Continue sharing
- **Stop**: End screen sharing
- Notification bar shows you're sharing

### View Shared Content

As a participant:
- Shared content appears in main window
- Your camera in small tile
- Click **"Pin"** to keep it large
- Click **"Hide self"** to maximize shared content

### Multiple Sharers

- Only one person shares at a time
- Others see "Someone is sharing"
- Must wait or host can change presenter

## Recording Meetings

### Start Recording

**Host**:
1. Click **‚ãÆ** (more) ‚Üí **"Record meeting"**
2. All participants notified of recording
3. Red **REC** indicator appears
4. Recording starts

**Permissions**:
- Only host can start recording (default)
- Or host enables "Allow participants to record"

### Recording Indicator

When meeting is recorded:
- üî¥ **Red REC badge** in corner
- All participants see indicator
- Can't hide recording status

### Stop Recording

1. Click **‚ãÆ** ‚Üí **"Stop recording"**
2. Or click **REC** badge ‚Üí **"Stop"**
3. Recording stops
4. Processing begins

### Access Recordings

**Cloud Recordings** (default):
1. Saved to NEXUS Drive automatically
2. Go to Drive ‚Üí **"Meet Recordings"** folder
3. Available within 1 hour after meeting
4. Kept for 90 days (can download for longer)

**Local Recordings**:
1. Enable in Settings ‚Üí Recordings ‚Üí "Save locally"
2. Saved to Downloads folder
3. File format: WebM or MP4

### Recording Files

Each recording includes:
- **Video file**: Meeting video and audio
- **Audio only file**: Audio track only (optional)
- **Chat transcript**: All chat messages
- **Transcript**: Auto-generated captions (if enabled)

### Manage Recordings

**Share Recording**:
1. Find recording in Drive
2. Right-click ‚Üí **"Share"**
3. Set permissions (Viewer recommended)
4. Copy link to share

**Download Recording**:
- Right-click ‚Üí **"Download"**
- Save to computer
- Edit with video software if needed

**Delete Recording**:
- Right-click ‚Üí **"Remove"**
- Moves to Trash
- Permanently deleted after 30 days

## Troubleshooting

### Can't Hear Audio?

**Check Microphone**:
- Verify microphone not muted in Meet
- Check system volume
- Select correct mic device
- Test mic in Settings

**Check Speakers**:
- Increase volume
- Select correct speaker device
- Test speakers in Settings
- Try headphones

**Browser Permissions**:
- Check mic/speaker allowed
- Look for blocked icon in address bar
- Allow permissions and refresh

### Video Not Working?

**Camera Issues**:
- Check camera turned on in Meet
- Verify browser has camera permission
- Close other apps using camera (Zoom, Skype, etc.)
- Try different camera if multiple

**Poor Video Quality**:
- Reduce resolution to 720p
- Close other bandwidth-heavy apps
- Move closer to Wi-Fi router
- Disable virtual background
- Turn off HD mode

### Echo or Feedback?

**Causes**:
- Multiple devices in same room
- Speakers too loud
- No headphones

**Solutions**:
- Mute all but one device
- Use headphones
- Reduce speaker volume
- Enable echo cancellation in Settings

### Connection Issues?

**Unstable Connection**:
- Check internet speed (need 1+ Mbps)
- Close bandwidth-heavy apps
- Switch from Wi-Fi to Ethernet
- Move closer to router
- Ask others to turn off video

**Can't Join Meeting**:
- Verify meeting link is correct
- Check if meeting locked
- Try different browser
- Clear browser cache
- Contact host for new link

### Screen Sharing Not Working?

**Permission Denied**:
- Browser needs screen capture permission
- macOS: System Preferences ‚Üí Security ‚Üí Screen Recording
- Allow browser to record screen
- Restart browser

**Can't See Shared Screen**:
- Refresh page
- Check internet connection
- Ask presenter to re-share
- Try different browser

## Tips and Best Practices

### 1. Before the Meeting

**Test Your Setup**:
- Test audio and video 5 minutes before
- Check lighting (face should be well-lit)
- Close unnecessary apps
- Connect to power (not battery)

**Environment**:
- Quiet location
- Professional background or use blur
- Good lighting
- Stable internet connection

### 2. During the Meeting

**Professional Etiquette**:
- Join on time
- Mute when not speaking
- Video on for better engagement
- Dress professionally (at least top half!)
- Look at camera when speaking

**Engagement**:
- Use reactions to participate
- Raise hand to speak
- Use chat for questions
- Stay present (don't multitask)

### 3. As a Host

**Preparation**:
- Create agenda
- Send meeting link in advance
- Start 5 minutes early
- Prepare screen shares beforehand

**During Meeting**:
- Introduce participants
- Manage muting for large groups
- Monitor chat for questions
- Keep meeting on track
- End on time

### 4. Technical Tips

**Bandwidth Optimization**:
- Turn off video when not needed
- Use audio-only mode for mobile data
- Limit number of visible participants
- Disable virtual background

**Security**:
- Use unique meeting codes
- Enable waiting room for external guests
- Lock meeting once all attendees joined
- Don't share meeting links publicly
