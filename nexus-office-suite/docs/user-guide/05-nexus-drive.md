## Uploading Files

### Upload Methods

#### Drag and Drop
1. Open NEXUS Drive in browser
2. Navigate to destination folder
3. Drag files from your computer
4. Drop into Drive window
5. Upload begins automatically

#### Upload Button
1. Click **"+ New"** button
2. Select **"File upload"**
3. Browse and select file(s)
4. Click **"Open"**
5. Files upload to current folder

#### Folder Upload
1. Click **"+ New"** ‚Üí **"Folder upload"**
2. Select folder from your computer
3. Entire folder structure uploads
4. Maintains folder hierarchy

#### Email to Drive
Send files via email:
1. Get your Drive email address:
   - Click **Settings** ‚Üí **"Get email address"**
2. Email files to that address
3. Files appear in **"Emailed to Drive"** folder

### Upload Progress

- Progress bar shows upload status
- Multiple files upload in parallel
- Can continue working while uploading
- Click "X" to cancel upload

### Supported File Types

**Documents**
- Documents: .doc, .docx, .odt, .pdf, .txt
- Spreadsheets: .xls, .xlsx, .ods, .csv
- Presentations: .ppt, .pptx, .odp

**Media**
- Images: .jpg, .png, .gif, .bmp, .svg, .webp
- Videos: .mp4, .mov, .avi, .mkv, .webm
- Audio: .mp3, .wav, .ogg, .m4a

**Archives**
- .zip, .rar, .7z, .tar, .gz

**Other**
- Code files: .html, .js, .py, .java, etc.
- Design files: .psd, .ai, .sketch, .fig
- 3D files: .obj, .stl, .dae

### File Size Limits

- Single file: Up to 5 GB via web
- Via desktop app: Up to 50 GB
- Total storage: Based on your plan

## Sharing Files and Folders

### Share with People

#### Share Individual File/Folder
1. Right-click file or folder
2. Select **"Share"**
3. Enter email addresses or names
4. Choose permission level:
   - **Viewer**: Can view and download
   - **Commenter**: Can view and comment
   - **Editor**: Can edit and organize
5. Add message (optional)
6. Click **"Send"**

#### Notify Recipients
- Email sent with link to file/folder
- Recipients get notification
- Shows in their "Shared with me"

### Permission Levels Explained

**Viewer**
- ‚úÖ View file
- ‚úÖ Download file
- ‚úÖ Copy file
- ‚ùå Edit file
- ‚ùå Share with others

**Commenter**
- ‚úÖ All Viewer permissions
- ‚úÖ Add comments
- ‚ùå Edit file
- ‚ùå Share with others

**Editor**
- ‚úÖ All Commenter permissions
- ‚úÖ Edit file
- ‚úÖ Rename, move, delete
- ‚úÖ Share with others (if allowed)
- ‚ùå Change owner

### Shareable Links

#### Get Link
1. Right-click file/folder
2. Select **"Get link"**
3. Set link access:
   - **Restricted**: Only invited people
   - **Anyone with the link**: Anyone who has URL
   - **Public**: Discoverable via search
4. Set permission (Viewer, Commenter, Editor)
5. Click **"Copy link"**

#### Link Expiration
1. In sharing dialog, click **"Link settings"**
2. Set **"Expires"** date
3. Link becomes invalid after date
4. Good for temporary sharing

#### Password Protection
1. Link settings ‚Üí **"Require password"**
2. Set password
3. Users must enter password to access
4. Share password separately for security

### Advanced Sharing Options

**Prevent Downloads**
- Viewers can't download, print, or copy
- Share ‚Üí Advanced ‚Üí "Disable download"
- Good for sensitive documents

**Prevent Editors from Changing Access**
- Only you can modify sharing
- Prevents editors from adding new people
- Share ‚Üí Advanced ‚Üí "Disable options"

**Request Access**
If file is restricted:
1. User clicks link
2. Sees "Request Access" button
3. Request sent to owner
4. Owner approves or denies

### Shared with Me

View all files others shared with you:
1. Click **"Shared with me"** in left sidebar
2. See files/folders shared by others
3. Organized by recency
4. Can move to your Drive to organize

### Remove Access

Stop sharing with someone:
1. Right-click shared file/folder
2. Select **"Share"** ‚Üí **"Advanced"**
3. Find person in list
4. Click "X" to remove
5. They lose access immediately

## File Versioning

### Automatic Versioning

NEXUS Drive automatically saves versions:
- Every edit creates new version
- Up to 100 versions kept
- 30-day retention minimum
- Important versions kept forever

### View Version History

1. Right-click file
2. Select **"Version history"**
3. Panel shows all versions
4. See:
   - Date and time
   - Who made changes
   - Size of file

### Restore Previous Version

1. Open version history
2. Click on a previous version
3. Preview the version
4. Click **"Restore"**
5. This version becomes current
6. Old current version is not lost (becomes a version)

### Download Specific Version

1. View version history
2. Click **‚ãÆ** next to version
3. Select **"Download"**

### Keep Version Forever

1. Version history
2. Click **‚ãÆ** next to version
3. Select **"Keep forever"**
4. Version won't be auto-deleted
5. Good for milestones, final versions

### Name a Version

1. Version history
2. Click **‚ãÆ** ‚Üí **"Name this version"**
3. Enter descriptive name
4. E.g., "Final Draft", "Client Review", "Published"

## Search and Filters

### Basic Search

1. Click search box (top)
2. Type search term
3. Results appear as you type
4. Press Enter for full results

### Search Types

**By Name**
- Type file name
- Partial matches work
- "report" finds "Q4_report.xlsx"

**By Content**
- Searches inside documents
- Only for supported formats (Docs, PDFs, etc.)
- "budget" finds files containing "budget"

**By File Type**
- Type "type:pdf" for PDFs
- "type:document" for documents
- "type:image" for images

**By Owner**
- "owner:john@company.com"
- "owner:me" for your files

**By Date**
- "after:2025/01/01"
- "before:2025/12/31"
- "yesterday", "last week", "this month"

### Advanced Search

Click **üîç** icon for advanced options:

**Location**
- My Drive
- Shared with me
- Specific folder

**File Type**
- Documents
- Spreadsheets
- Presentations
- Images
- Videos
- PDFs
- Folders

**Date Modified**
- Today
- Last 7 days
- Last 30 days
- Custom range

**Ownership**
- Owned by me
- Not owned by me
- Specific person

**Sharing**
- Shared with me
- Shared by me
- Not shared

**Status**
- Starred
- Trashed

### Save Search

1. Perform search
2. Click **"Save search"**
3. Name your search
4. Appears in left sidebar
5. Click to run again

### Filters

Use toolbar filters:
- **Type**: Filter by file type

## Storage Management

### Check Storage Usage

1. Click **Settings** ‚Üí **"Storage"**
2. See breakdown:
   - NEXUS Drive: Files and folders
   - NEXUS Writer: Documents
   - NEXUS Sheets: Spreadsheets
   - NEXUS Slides: Presentations
   - Trash: Deleted items
3. Visual chart shows usage

### Free Up Space

#### Empty Trash
1. Click **"Trash"** in left sidebar
2. Click **"Empty trash"**
3. Permanently deletes all trashed items
4. Cannot be undone

#### Delete Large Files
1. Settings ‚Üí Storage ‚Üí **"Review large files"**
2. Sorted by size
3. Select files to delete
4. Click **"Delete"**

#### Remove Duplicate Files
1. Settings ‚Üí Storage ‚Üí **"Find duplicates"**

3. Review duplicates
4. Delete extras

#### Compress Files
- Right-click files ‚Üí **"Download"**
- Select multiple files
- Downloads as .zip
- Upload compressed version
- Delete originals

### Upgrade Storage

1. Click **"Upgrade storage"** button
2. Choose plan:
   - **100 GB**: $1.99/month
   - **1 TB**: $9.99/month
   - **2 TB**: $19.99/month

### Shared Drive Space

Files don't count against your quota if:
- Owned by someone else
- Shared with you
- In a Team Drive

## Tips and Best Practices

### 1. Organization Tips

**Folder Structure**
- Use main categories (Work, Personal, Archive)
- Keep folders 3 levels deep maximum
- Name folders clearly and consistently

**Naming Convention**
- Use descriptive names
- Include dates: YYYY-MM-DD format
- Avoid special characters (except - and _)
- Example: "2025-01-15_Q4-Report_Final.pdf"

**Regular Cleanup**
- Review files monthly
- Archive old files
- Delete duplicates
- Empty trash regularly

### 2. Sharing Best Practices

**Security**
- Use "Restricted" for sensitive files
- Set link expiration dates
- Use password protection when needed
- Review sharing permissions quarterly

**Collaboration**
- Give appropriate permission levels
- Use "Editor" only when necessary
- Prefer "Commenter" for reviews
- Document sharing decisions

### 3. Storage Optimization

**File Management**
- Compress large files before uploading
- Delete old versions if not needed
- Use Team Drives for team files (doesn't count against quota)
- Archive infrequently accessed files

**Efficiency**
- Upload folders, not individual files
- Use desktop app for large transfers
- Schedule uploads during off-hours
- Monitor storage usage monthly

### 4. Backup Strategy

**3-2-1 Rule**
- 3 copies of data
- 2 different media types
- 1 offsite copy (NEXUS Drive counts!)

**Critical Files**
- Keep important versions forever
- Download local backups periodically
- Use Team Drive for critical business files

## Troubleshooting

### Upload failed?
- Check file size (under 5 GB)
- Verify internet connection
- Check storage quota
- Try again with smaller batches
- Use desktop app for large files

### Can't find a file?
- Check Trash folder
- Use search with filters
- Check "Shared with me"
- Ask file owner if removed

### Sync not working?
- Restart desktop app
- Check internet connection
- Verify selective sync settings
- Pause and resume sync
- Check for app updates

### Out of storage?
- Empty trash
- Delete large files
- Remove duplicates
- Upgrade storage plan

### Sharing not working?
- Verify email address is correct
- Check if user has NEXUS account
- Try link sharing instead
- Contact admin for external sharing
