### Create a New Document

## Text Formatting

### Basic Text Formatting

#### Bold, Italic, Underline

**Method 1: Toolbar**
- Select text
- Click **B** (Bold), **I** (Italic), or **U** (Underline) in the toolbar

**Method 2: Keyboard Shortcuts**
- **Bold**: `Ctrl+B` or `Cmd+B`
- **Italic**: `Ctrl+I` or `Cmd+I`
- **Underline**: `Ctrl+U` or `Cmd+U`

**Method 3: Markdown Syntax** (while typing)
- **Bold**: Type `**text**` or `__text__`
- **Italic**: Type `*text*` or `_text_`
- **Bold+Italic**: Type `***text***`

#### Font and Size

**Change Font**
1. Select text
2. Click the **Font dropdown** (default: Arial)
3. Choose from available fonts:
   - Arial
   - Times New Roman
   - Helvetica
   - Georgia
   - Courier New
   - Comic Sans MS
   - And more...

**Change Size**
1. Select text
2. Click the **Size dropdown** (default: 11pt)
3. Choose size from 8pt to 72pt
4. Or type a custom size

#### Text Color and Highlighting

**Text Color**
1. Select text
2. Click the **Text Color** button (A with color bar)
3. Choose a color from the palette
4. Or enter a hex code (e.g., #FF5733)

**Highlight Color**
1. Select text
2. Click the **Highlight** button (marker icon)
3. Choose a highlight color
4. Click again to remove highlighting

### Paragraph Formatting

#### Headings

Use headings to structure your document:

**Method 1: Styles Dropdown**
1. Place cursor in paragraph
2. Click **Styles** dropdown
3. Select Heading 1, Heading 2, or Heading 3

**Method 2: Keyboard Shortcuts**
- **Heading 1**: `Ctrl+Alt+1`
- **Heading 2**: `Ctrl+Alt+2`
- **Heading 3**: `Ctrl+Alt+3`
- **Normal text**: `Ctrl+Alt+0`

**Method 3: Markdown Syntax**
- Type `# ` for Heading 1
- Type `## ` for Heading 2
- Type `### ` for Heading 3

#### Alignment

Align text left, center, right, or justify:

**Toolbar Buttons**
- Click alignment buttons in the toolbar

**Keyboard Shortcuts**
- **Left align**: `Ctrl+Shift+L`
- **Center align**: `Ctrl+Shift+E`
- **Right align**: `Ctrl+Shift+R`
- **Justify**: `Ctrl+Shift+J`

#### Line Spacing

1. Select text or place cursor in paragraph
2. Click **Format** â†’ **Line Spacing**
3. Choose:
   - Single (1.0)
   - 1.15 (default)
   - 1.5
   - Double (2.0)
   - Custom

#### Lists

**Bulleted List**
- Click the **Bullets** button
- Or press `Ctrl+Shift+8`
- Or type `- ` or `* ` at the start of a line

**Numbered List**
- Click the **Numbering** button
- Or press `Ctrl+Shift+7`
- Or type `1. ` at the start of a line

**Increase/Decrease Indent**
- `Tab` to increase indent (create sub-items)
- `Shift+Tab` to decrease indent

#### Indentation

1. Select paragraphs
2. Click **Format** â†’ **Indentation**
3. Set:
   - **Left indent**: Distance from left margin
   - **Right indent**: Distance from right margin
   - **First line**: Special indent for first line
   - **Hanging**: Indent all lines except first

1. Click **Insert** â†’ **Image** â†’ **Upload from Computer**
2. Select image file (JPG, PNG, GIF, SVG)
3. Click **"Open"**
4. Image is inserted at cursor position

#### Insert from Drive

#### Resize and Position Images

  - **Alignment**: Left, center, right
  - **Size**: Enter exact dimensions
  - **Alt text**: For accessibility

### Tables

#### Create a Table
1. Click **Insert** â†’ **Table**
2. Hover over the grid to select size (e.g., 3Ã—4)
3. Click to insert
4. Or click **"Insert custom table"** for exact dimensions

#### Table Operations

**Add/Remove Rows**

- Or **Delete row**

**Add/Remove Columns**
- Right-click in table
- Select **Insert column left/right**

- Right-click â†’ **Merge cells**

  - Alignment

### Links

#### Insert a Hyperlink
1. Select text to link
2. Click **Insert** â†’ **Link**
3. Or press `Ctrl+K`

5. Click **"Apply"**

#### Link to Another Document
1. Select text
2. Click **Insert** â†’ **Link**
3. Click **"Search your documents"**
4. Find and select the document
5. Click **"Apply"**

#### Edit or Remove Link
- Click the link
- Click **"Edit"** to change URL
- Click **"Remove"** to unlink

### Page Breaks

Insert a page break to start content on a new page:

1. Place cursor where you want the break
2. Click **Insert** â†’ **Page Break**
3. Or press `Ctrl+Enter`

### Table of Contents

Automatically generate a table of contents from headings:

1. Place cursor where you want the TOC
2. Click **Insert** â†’ **Table of Contents**
3. Choose style:
   - With page numbers
   - With links
   - Plain text
4. TOC updates automatically as you add/remove headings

> **ðŸ’¡ Tip**: Click **"Update"** on the TOC to refresh it manually

### Equations

Insert mathematical equations:

1. Click **Insert** â†’ **Equation**
2. Use the equation editor:
   - Type using LaTeX syntax (e.g., `\frac{a}{b}`)
   - Or use the visual editor
3. Click outside to finish

**Examples**:
- Fraction: `\frac{numerator}{denominator}`
- Square root: `\sqrt{x}`
- Summation: `\sum_{i=1}^{n} x_i`
- Integral: `\int_{a}^{b} f(x)dx`

## Version History

Never lose your work with automatic version history.

### View Version History

1. Click **File** â†’ **Version History**
2. Or press `Ctrl+Alt+Shift+H`
3. See timeline of all changes

### Version History Panel

The panel shows:
- **Timestamp**: When the change was made
- **User**: Who made the change
- **Changes**: Summary of what changed
- **Restore**: Option to revert to this version

### Restore a Previous Version

1. Open Version History
2. Click on a previous version
3. Review the changes
4. Click **"Restore this version"**
5. Confirm the restoration

> **âš ï¸ Warning**: Restoring creates a new version; the current version is not lost.

### Name a Version

Create named checkpoints:

1. Click **File** â†’ **Version History** â†’ **Name current version**
2. Enter a name (e.g., "Final Draft", "Pre-review")
3. Click **"Save"**
4. Named versions appear prominently in the history

## Sharing and Permissions

### Share a Document

1. Click the **"Share"** button (top-right)
2. Enter email addresses or names
3. Set permission level:
   - **Viewer**: Can only view
   - **Commenter**: Can view and comment
   - **Editor**: Can edit the document
4. Add a message (optional)
5. Click **"Send"**

### Get Shareable Link

1. Click **"Share"** â†’ **"Get Link"**
2. Choose link access:
   - **Restricted**: Only specific people
   - **Anyone with the link**: Anyone who has the URL
   - **Public**: Discoverable via search
3. Set permission (Viewer, Commenter, Editor)
4. Click **"Copy link"**
5. Share the link via email, chat, etc.

### Change Permissions

1. Click **"Share"**
2. Find the person in the list
3. Click their permission level
4. Select new permission or **"Remove access"**

### View Who Has Access

1. Click **"Share"**
2. See all users with access
3. See their permission levels
4. See when they last viewed

### Advanced Sharing Options

- **Prevent editors from changing access**: Only you can share
- **Prevent viewers from downloading**: View-only mode
- **Disable comments**: Turn off commenting
- **Expiration date**: Auto-remove access after date

## Tips and Best Practices

### 1. Use Styles for Consistency
- Apply Heading styles instead of manual formatting
- Ensures consistency throughout the document
- Enables automatic table of contents

### 2. Collaborate Effectively
- Use comments for feedback
- Use suggesting mode when reviewing
- @mention team members for attention
- Resolve comments when addressed

### 3. Organize with Structure
- Use headings to create document structure
- Insert table of contents for long documents
- Use page breaks for logical sections

### 4. Keep Track of Versions
- Name important versions (e.g., "Final Draft")
- Review version history before major changes
- Use version history to see who changed what

### 5. Share Securely
- Use appropriate permission levels
- Avoid "Anyone with the link" for sensitive docs
- Set expiration dates for temporary access
- Review who has access regularly

## Troubleshooting

### Document not saving?
- Check internet connection
- Look for "Saving..." in the title bar
- If stuck, copy content and refresh page
- Contact support if problem persists

### Can't see collaborator's changes?
- Refresh the page (F5)
- Check if document is in offline mode
- Verify collaborator has edit permissions
- Clear browser cache

### Formatting looks different?
- Different browsers may render slightly differently
- Export to PDF for consistent appearance
- Check if custom fonts are installed
