# Basic Usage Training

**Duration**: 1 hour
**Audience**: New Users

---

## Module 1: Getting Started (15 min)

### Creating Your Account
1. Visit NEXUS platform
2. Click "Sign Up"
3. Fill in details
4. Verify email
5. Log in

### Navigating the Hub
- Dashboard overview
- Application launcher
- Search functionality
- Notifications
- Profile settings

**Exercise**: Create account and explore the Hub

---

## Module 2: Creating Documents (15 min)

### NEXUS Writer
1. Click "New Document"
2. Add title
3. Format text (bold, italic, headings)
4. Insert image
5. Save automatically

**Exercise**: Create a simple memo

---

## Module 3: Building Spreadsheets (15 min)

### NEXUS Sheets
1. Create new spreadsheet
2. Enter data
3. Use basic formulas (SUM, AVERAGE)
4. Create a chart
5. Format cells

**Exercise**: Create budget spreadsheet

---

## Module 4: Sharing and Collaboration (15 min)

### Sharing Files
1. Click "Share" button
2. Add email addresses
3. Set permissions (View/Edit)
4. Send invitation

### Real-Time Collaboration
- See other users' cursors
- Add comments
- Track changes

**Exercise**: Share a document with a colleague

---

## Quiz

1. How do you create a new document?
2. Where do you find shared files?
3. How do you add someone as editor?
4. What keyboard shortcut opens search?

**Answers**: 1) Click "New Document", 2) "Shared with me", 3) Share → Enter email → Editor, 4) Ctrl+K
